<span style={{ fontSize: '24px', fontWeight: 'bold' }}>napi-rs</span> is a minimal library for building compiled <span style={{ color: 'green', fontWeight: 'bold' }}>NodeJS</span> add-ons in <span style={{ color: '#b7410e', fontWeight: 'bold' }}>Rust</span>.

```rust title=lib.rs lineNumbers=true
#[js_function(1)]
fn fibonacci(ctx: CallContext) -> Result<JsNumber> {
  let n = ctx.get::<JsNumber>(0)?.try_into()?;
  ctx.env.create_int64(fibonacci_native(n))
}
```

```js title=index.js lineNumbers=true
function hello() {
  let result = fibonacci(10000);
  console.log(result);
  return result;
}
```
